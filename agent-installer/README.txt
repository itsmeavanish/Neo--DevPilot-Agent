============================================
    JARVIS Remote Agent - Quick Start
============================================

STEP 1: Install the Agent
-------------------------
1. Double-click "install.bat"
2. Enter a name for your laptop
3. Done! A shortcut will be created on your desktop.

STEP 2: Start the Agent
-----------------------
Double-click "JARVIS Agent" on your desktop.

You will see a PAIRING CODE like this:

   ==================================================
          YOUR PAIRING CODE:

             >>> A1B2C3 <<<

          Enter this code in your JARVIS app
   ==================================================

STEP 3: Pair Your Phone
-----------------------
1. Install the JARVIS app on your phone
2. Enter the pairing code shown on your laptop
3. You're connected! Only YOUR phone can control YOUR laptop.

============================================
             IMPORTANT NOTES
============================================

- The pairing code is UNIQUE to your laptop
- Only the phone with this code can access your laptop
- Other users cannot see or control your laptop
- Keep the agent window open while using JARVIS

============================================
             TROUBLESHOOTING
============================================

"Python not found"
  -> Install Python from https://python.org
  -> Make sure to check "Add to PATH" during install

"Connection failed"
  -> Check your internet connection
  -> Make sure the JARVIS server is running

"Pairing code not working"
  -> Make sure you entered the code exactly as shown
  -> Restart the agent and try the new code

============================================
              REQUIREMENTS
============================================

- Windows 10/11
- Python 3.8 or higher
- Internet connection
